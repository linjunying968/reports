🔬 v2 方法论：数据怎么变成 λ？

第〇步 · 数据采集（分析开始前必做）

本届赛事每队已打完的比赛比分、进球、失球
近期战绩赛前最后 3-5 场比分（热身赛/预选赛）
关键统计射正数、零封率、BTTS 率（如有）

第一步 · 独立 λ 估算（不看赔率）

公式λ = (球队近期场均进球 + 对手近期场均失球) ÷ 2
西班牙λ= (1.0 + 1.0)/2 = 1.00 ← 远低于市场假设的 2.45
乌拉圭λ= (1.0 + 0.0)/2 = 0.60 ← 远低于市场假设的 1.75
埃及λ= (1.0 + 2.0)/2 = 1.50 ≈ 市场假设的 1.70 ✓
调整上一场 0 射正 → λ×0.75；连续 3+ 场进球 ≥2 → λ×1.15

第二步 · Poisson 跑分

输入第一步的独立 λ（不校准市场）
输出胜/平/负概率、大小球、BTTS、比分网格

第三步 · 市场对比找分歧（最后才看赔率）

埃及数据 80.2% vs 市场 84.0% → 🟢 基本一致
结论分歧大的场次 = 价值所在。信数据不信市场。

不限制1X2、大小球、BTTS 任意混合
v1 的根本问题：λ 校准到市场赔率 → 市场说西班牙 86.4%胜 → λ 算出来 2.45 → Poisson 再跑一次还是 86.4%。这是把市场的判断套了一层数学壳，不是独立分析。
